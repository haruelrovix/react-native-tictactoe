# react-native-tictactoe
A Tic Tac Toe game from scratch, using React Native.

### Usage

Make sure you have setup your environment to run React Native project. 

1. Install dependencies:

```
yarn
```

2. Then, for Android, run:

```
yarn android
```

3. For iOS, run:  

```
yarn ios
```
